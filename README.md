# サンプルデータの説明(csv, tsv, xlsx, sas7bdatフォルダ内)
- Sales ・・・購買ログ
    - UserID: ユーザー ID
    - ProductID: 商品 ID
    - Timestamp: 購買時刻
- UserMaster ・・・属性データ
    - UserID: ユーザー ID
    - Age: 年齢
    - Sex: 性別
    - Pref: 都道府県
- Products ・・・商品マスタデータ
    - ProductID: 商品 ID
    - ProductName: 商品名
    - Price: 価格
    - Category: 商品ジャンル
    - CreatedDate: 商品が作られた年月日
